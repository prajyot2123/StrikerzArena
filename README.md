# CricAura - Cricket Auction Platform

Welcome to the **CricAura** project repository. This is a premium cricket auction platform featuring real-time bidding, AI-powered player classification, and comprehensive tournament management.

## 📂 Project Structure

- **`backend/`**: Node.js/Express server with MongoDB.
- **`frontend/`**: React/Vite dashboard for players, organizers, and super admins.
- **`ml-service/`**: Python/Flask service for player classification.
